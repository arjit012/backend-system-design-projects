# backend-system-design-projects

This repository contains small backend POCs for common system design scenarios such as idempotent order creation and similar reliability-focused patterns.

The main focus of each project is the core problem statement and the design approach behind it. Because of that, some secondary production details may be intentionally lightweight or left out, such as exhaustive validation, edge-case handling, or full test coverage.

All projects in this repository follow the same boilerplate so it is easier to compare solutions, understand the pattern being demonstrated, and build new scenarios on top of a familiar structure.
